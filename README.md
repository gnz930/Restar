# Shiharaikun

Minimal iOS payment tracker.

## Requirements
- Xcode 15+
- iOS 16+

## Setup
1. (Optional) If you edit `project.yml`, regenerate the project:
   `xcodegen`
2. Open `Shiharaikun.xcodeproj` in Xcode
3. Set your Signing Team and Bundle ID if needed
4. Build and run on your iPhone (free Apple ID works, re-sign every 7 days)

## Features
- Add/edit recurring payments
- Auto next due date + annualized totals
- Mark paid to shift next due date
- Pause/resume items
